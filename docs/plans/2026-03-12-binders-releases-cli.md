# Binders, Versioned Releases, and CLI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Produce versioned “binders” (server binary + UI assets + scripts + config) for every release, store them in a dedicated repository under plain-version tags (e.g. `1.0.1`), and provide a CLI to download and run a version; binaries and runtime are configurable via YAML.

**Architecture:** A dedicated **binders repo** holds release workflow and artifacts. On tag push (e.g. `1.0.1`), CI clones backend and frontend repos at that tag, builds server binary and UI, assembles a fixed-name top-level folder `sessiondb/` (README, setup.sh, .env.example, server/, ui/), and publishes a single full zip to the GitHub Release. A small **CLI** (Go) can download a release by version and run setup; runtime options (ports, paths, env file) are read from an optional **YAML config** (e.g. `sessiondb.yaml`).

**Tech Stack:** Go (backend, CLI), Node/Vite (frontend), shell scripts, YAML (runtime config), GitHub Actions (binders repo), GitLab CI (existing BE/FE).

---

## Prerequisites and Conventions

- **Backend repo:** `sessiondb/service` (or your backend repo root). Build: `go build -o sessiondb ./cmd/server`. Config today: TOML + .env (viper).
- **Frontend repo:** Separate repo with Vite app; build: `npm ci && npm run build` → `dist/`.
- **Binders repo:** New repo (e.g. `sessiondb/binders` or `sessiondb/releases`). Holds no application source; only release workflow and generated artifacts. Release tags are **plain versions**: `1.0.1`, not `sessiondb-1.0.1`.
- **Full zip layout:** Unzipping `1.0.1-full.zip` yields one top-level directory **`sessiondb/`** (fixed name) with:
  - `sessiondb/README.md` — version and usage guidelines
  - `sessiondb/setup.sh` — run server + UI (sources .env, reads optional sessiondb.yaml)
  - `sessiondb/.env.example` — all env vars for backend and frontend
  - `sessiondb/sessiondb.yaml` — optional runtime config (ports, paths, env file)
  - `sessiondb/server/` — server binary + run script
  - `sessiondb/ui/` — built frontend (e.g. dist/ + serve script)

### Task 5: Binders repo CI — build and publish full zip on tag

**Files:**
- Create: `.github/workflows/release.yml` (or `.gitlab-ci.yml` in binders repo)

**Step 1: Add GitHub Actions workflow**

Create `.github/workflows/release.yml` in the binders repo. Use plain version tags (e.g. `1.0.1`). Job: on `push` tags matching `*.*.*`, clone backend repo at tag, clone frontend repo at tag, build both, assemble `sessiondb/`, zip, create GitHub Release, upload zip.

Replace `YOUR_ORG/backend-repo` and `YOUR_ORG/frontend-repo` with real repo names.

```yaml
name: Release Binder

on:
  push:
    tags:
      - '[0-9]+.[0-9]+.[0-9]+'

jobs:
  release:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout binders repo
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Set version from tag
        id: version
        run: echo "VERSION=${GITHUB_REF#refs/tags/}" >> $GITHUB_OUTPUT

      - name: Clone and build backend
        run: |
          git clone --depth 1 --branch ${{ steps.version.outputs.VERSION }} https://github.com/YOUR_ORG/backend-repo.git backend-src
          cd backend-src
          go build -o sessiondb ./cmd/server
          mkdir -p ../artifacts/server
          cp sessiondb ../artifacts/server/
          cp ../scripts/server-run.sh ../artifacts/server/run.sh
          chmod +x ../artifacts/server/run.sh

      - name: Clone and build frontend
        run: |
          git clone --depth 1 --branch ${{ steps.version.outputs.VERSION }} https://github.com/YOUR_ORG/frontend-repo.git frontend-src
          cd frontend-src
          npm ci
          npm run build
          mkdir -p ../artifacts/ui
          cp -r dist ../artifacts/ui/
          cp ../scripts/ui-serve.sh ../artifacts/ui/serve.sh
          chmod +x ../artifacts/ui/serve.sh

      - name: Assemble sessiondb folder
        run: |
          VERSION="${{ steps.version.outputs.VERSION }}"
          mkdir -p sessiondb
          cp -r artifacts/server sessiondb/
          cp -r artifacts/ui sessiondb/
          cp scripts/setup.sh sessiondb/
          chmod +x sessiondb/setup.sh
          cp templates/env.example sessiondb/.env.example
          cp templates/sessiondb.yaml sessiondb/
          sed "s/{{VERSION}}/$VERSION/g" templates/README-release.md > sessiondb/README.md

      - name: Create zip
        run: |
          VERSION="${{ steps.version.outputs.VERSION }}"
          zip -r "${VERSION}-full.zip" sessiondb

      - name: Create GitHub Release and upload
        uses: softprops/action-gh-release@v2
        with:
          files: ${{ steps.version.outputs.VERSION }}-full.zip
          generate_release_notes: true
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Fix paths in the workflow so that `scripts/` and `templates/` are at repo root (adjust `cp` paths if your layout differs). If backend/frontend are private, use a token and `https://x-access-token:TOKEN@github.com/...` for clone.

```go
package main

import (
	"os"

	"gopkg.in/yaml.v3"
)

type SessionDBConfig struct {
	Server struct {
		Port   string `yaml:"port"`
		Binary string `yaml:"binary"`
	} `yaml:"server"`
	UI struct {
		Port int    `yaml:"port"`
		Dir  string `yaml:"dir"`
	} `yaml:"ui"`
	EnvFile string `yaml:"env_file"`
}

func loadConfig(path string) (*SessionDBConfig, error) {
	data, err := os.ReadFile(path)
	if err != nil {
		return nil, err
	}
	var c SessionDBConfig
	if err := yaml.Unmarshal(data, &c); err != nil {
		return nil, err
	}
	return &c, nil
}
```

**Step 3: Implement get (download + extract)**

Create `get.go`:

```go
package main

import (
	"archive/zip"
	"fmt"
	"io"
	"net/http"
	"os"
	"path/filepath"
)

const releasesURL = "https://github.com/YOUR_ORG/binders/releases/download"

func get(version string, destDir string) error {
	url := fmt.Sprintf("%s/%s/%s-full.zip", releasesURL, version, version)
	resp, err := http.Get(url)
	if err != nil {
		return err
	}
	defer resp.Body.Close()
	if resp.StatusCode != http.StatusOK {
		return fmt.Errorf("download failed: %s", resp.Status)
	}
	zipPath := filepath.Join(destDir, "release.zip")
	out, err := os.Create(zipPath)
	if err != nil {
		return err
	}
	defer out.Close()
	if _, err := io.Copy(out, resp.Body); err != nil {
		return err
	}
	// Unzip to destDir/sessiondb
	r, err := zip.OpenReader(zipPath)
	if err != nil {
		return err
	}
	defer r.Close()
	for _, f := range r.File {
		dst := filepath.Join(destDir, f.Name)
		if f.FileInfo().IsDir() {
			_ = os.MkdirAll(dst, 0755)
			continue
		}
		_ = os.MkdirAll(filepath.Dir(dst), 0755)
		rc, err := f.Open()
		if err != nil {
			return err
		}
		w, err := os.Create(dst)
		if err != nil {
			rc.Close()
			return err
		}
		_, err = io.Copy(w, rc)
		w.Close()
		rc.Close()
		if err != nil {
			return err
		}
		if f.Mode()&0111 != 0 {
			_ = os.Chmod(dst, 0755)
		}
	}
	_ = os.Remove(zipPath)
	fmt.Printf("Downloaded and extracted %s to %s/sessiondb\n", version, destDir)
	return nil
}
```

**Step 4: Implement run (get + execute setup.sh)**

Create `run.go`:

```go
package main

import (
	"fmt"
	"os"
	"os/exec"
	"path/filepath"
)

func run(version string, workDir string) error {
	sessiondbDir := filepath.Join(workDir, "sessiondb")
	setup := filepath.Join(sessiondbDir, "setup.sh")
	if _, err := os.Stat(setup); os.IsNotExist(err) {
		if err := get(version, workDir); err != nil {
			return err
		}
	}
	cmd := exec.Command(setup)
	cmd.Dir = sessiondbDir
	cmd.Stdout = os.Stdout
	cmd.Stderr = os.Stderr
	cmd.Stdin = os.Stdin
	return cmd.Run()
}
```

**Step 5: Main entrypoint**

Create `main.go`:

```go
package main

import (
	"fmt"
	"os"
	"path/filepath"
)

func main() {
	if len(os.Args) < 2 {
		fmt.Fprintln(os.Stderr, "Usage: sessiondb <get|run> <version> [workdir]")
		os.Exit(1)
	}
	cmd := os.Args[1]
	version := ""
	if len(os.Args) > 2 {
		version = os.Args[2]
	}
	workDir := "."
	if len(os.Args) > 3 {
		workDir = os.Args[3]
	}
	workDir, _ = filepath.Abs(workDir)
	var err error
	switch cmd {
	case "get":
		if version == "" {
			fmt.Fprintln(os.Stderr, "Usage: sessiondb get <version> [workdir]")
			os.Exit(1)
		}
		err = get(version, workDir)
	case "run":
		if version == "" {
			fmt.Fprintln(os.Stderr, "Usage: sessiondb run <version> [workdir]")
			os.Exit(1)
		}
		err = run(version, workDir)
	default:
		fmt.Fprintf(os.Stderr, "Unknown command: %s\n", cmd)
		os.Exit(1)
	}
	if err != nil {
		fmt.Fprintln(os.Stderr, err)
		os.Exit(1)
	}
}
```
